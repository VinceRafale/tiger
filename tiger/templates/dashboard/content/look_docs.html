Overview

The HTML tab on the Look & Feel toolbar gives you a high level of control over the layout of your Takeout Tiger website. We are going to assume that, if you are reading this, you are familiar with HTML and CSS.

When you open the HTML tab, what you are looking at is more or less the contents of the <body> tag. You can do what you like with this markup, with two caveats:

  1. We automatically strip out any potentially malicious markup — <style> tags, <script> tags, inline JavaScript, etc. It's not that we don't trust you. We just don't necessarily trust the person who told you that embed code is safe.
  2. There are some big blocks of markup that, at least for now, you can't touch. Instead, you place them within your markup using a {% templatetag openvariable %}blockname{% templatetag closevariable %} syntax.

Block variables

{% templatetag openvariable %}logo{% templatetag closevariable %}
An <h2> tag containing the name of your restaurant. If you've uploaded a logo, CSS will be generated to set text-indent to -9999px and will set the logo as the background image.
{% templatetag openvariable %}search{% templatetag closevariable %}
A menu search form for your site.
{% templatetag openvariable %}menu{% templatetag closevariable %}
An unordered list tag containing all the main menu items as list elements.
{% templatetag openvariable %}messages{% templatetag closevariable %}
An unordered list tag containing messages for the site visitor (menu item added to order successfully, restaurant currently closed, etc.) as list elements.
{% templatetag openvariable %}cart{% templatetag closevariable %}
Block of text showing the number of items in the current order with a link to complete the order, or a message encouraging the view to place an online order. Only shown if online ordering is enabled.
{% templatetag openvariable %}sections{% templatetag closevariable %}
List of menu sections. Only displayed when viewing a menu section or menu item.
{% templatetag openvariable %}contact{% templatetag closevariable %}
Hours, address, and phone/fax for the restaurant.
{% templatetag openvariable %}pdf{% templatetag closevariable %}
Link to the PDF menu set to "featured". Only displayed if the restaurant has a featured PDF menu.
{% templatetag openvariable %}social{% templatetag closevariable %}
Displays "follow us on Twitter" / "become our fan on Facebook" depending on whether those services have been configured.
{% templatetag openvariable %}mail{% templatetag closevariable %}
A "join our mailing list" form. Displayed only if configured to do so on the Marketing > Integrations tab.
{% templatetag openvariable %}footer{% templatetag closevariable %}
Restaurant contact info plus Takeout Tiger branding and restaurant login links.
