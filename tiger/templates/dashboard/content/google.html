Click "Alternative methods" tab, and on that tab select the "meta tag" option (the first one). Copy and paste that meta tag into the field below.

Analytics

To add Google Analytics to your site, create a Google Analytics account and create a profile for your site (if you are using a custom domain, be sure to use the "www.yourdomain.com" rather than "yourdomain.com"). After creating the profile, you are taken to a page with a cut & paste code snippet. The Web Property ID is what you provide in the text box below.
